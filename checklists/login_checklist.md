# Login Page - Test Checklist

## Functional
- [ ] User can log in with valid credentials
- [ ] Login fails with invalid password
- [ ] Login fails with invalid email
- [ ] Login fails when email field is empty
- [ ] Login fails when password field is empty
- [ ] Validation messages are displayed correctly
- [ ] Error messages are user-friendly and readable

## UI / UX
- [ ] Email and password fields are clearly labeled
- [ ] Password field masks input characters
- [ ] Login button is disabled or validation is triggered when fields are empty
- [ ] Error messages are displayed near the relevant fields

## Security (basic awareness)
- [ ] Password is not visible in plain text
- [ ] Error message does not reveal whether email or password is incorrect
