# Registration Page - Test Checklist

## Functional
- [ ] User can register with valid data
- [ ] Registration fails when required fields are empty
- [ ] Registration fails when email format is invalid
- [ ] Registration fails when email is already registered
- [ ] Password validation rules are enforced
- [ ] Password confirmation is required
- [ ] Validation messages are displayed correctly

## Password Policy
- [ ] Minimum password length is enforced
- [ ] Uppercase letter requirement is enforced
- [ ] Lowercase letter requirement is enforced
- [ ] Special character requirement is enforced

## UI / UX
- [ ] Required fields are clearly marked
- [ ] Validation messages are displayed near the relevant fields
- [ ] Password requirements are visible to the user

## Security (basic awareness)
- [ ] Password is masked during input
- [ ] Validation messages do not expose sensitive information
